# Základy SQL - SELECT

Napíš SQL dotaz, ktorý vyberie všetkých študentov z tabuľky `students`, ktorí majú viac ako 18 rokov.

Výsledok by mal obsahovať stĺpce `id`, `name` a `age`, zoradené podľa veku zostupne.

## Tabuľka `students`

| id | name | age |
|----|------|-----|
| 1  | Ján  | 20  |
| 2  | Mária| 17  |
| 3  | Peter| 22  |
| 4  | Eva  | 19  |
| 5  | Tomáš| 16  |
